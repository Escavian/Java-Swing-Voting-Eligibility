# Java Swing Voting Eligibility App

A simple Java Swing GUI application that asks the user for their **name** and **age**, then displays whether they are eligible to vote in the United States.

## 📌 Features
- Clean and simple **Swing GUI**
- Text fields for entering **name** and **age**
- Button to check voting eligibility
- Displays a personalized message based on the user's input
- Basic input validation

## 🖥️ How It Works
1. The user enters their **name**
2. The user enters their **age**
3. The app checks if the age is **18 or older**
4. A message appears telling the user whether they can vote

## 🧰 Technologies Used
- Java
- Swing (JFrame, JLabel, JTextField, JButton)
- Basic conditional logic

## 🚀 How to Run
1. Clone or download this repository  
2. Open the project in your preferred IDE (Eclipse, IntelliJ, VS Code, etc.)
3. Run the main `.java` file  
4. The GUI window will appear
